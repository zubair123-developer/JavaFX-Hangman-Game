# JavaFX Hangman Desktop Application

A visually engaging, interactive desktop Hangman game built using the JavaFX framework. This project demonstrates event-driven programming, graphical user interface (GUI) layout management, and robust object-oriented logic in Java.

## 🚀 Features
* **Graphical User Interface (GUI):** Designed with modern JavaFX layout containers, featuring interactive keyboard buttons and dynamic text rendering.
* **Event-Driven Architecture:** Handled responsive user inputs, button state tracking, and live display text updates smoothly.
* **Robust Game Logic:** Separates core data processing (word generation, character masking, match checking) cleanly from the UI layout layer.
* **State Management:** Includes live tracking of remaining attempts, alpha-button disabling upon click, and explicit win/loss states.

## 🛠️ Project Structure
* **`module-info.java`** - Configures the Java module descriptors and framework dependencies for JavaFX runtime execution.
* **`hello-view.fxml`** - Establishes the XML-based structural design layout of the application UI components.
* **`pom.xml`** - Manages Project Object Model configurations, build plugins, and dependencies for Maven.
